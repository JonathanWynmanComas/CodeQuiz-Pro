# CodeQuiz Pro ⏱️

## Description
CodeQuiz Pro is a timed, interactive coding quiz built using vanilla JavaScript, HTML, and CSS.  
Users answer multiple-choice questions related to JavaScript fundamentals while racing against the clock.  
Incorrect answers reduce the remaining time, adding an extra challenge.

This project demonstrates core web development skills including DOM manipulation, event delegation, timers, and local storage.

---

## Features
- Timed multiple-choice quiz
- Dynamic question rendering
- Time penalty for incorrect answers
- Score calculation
- High score storage using Local Storage
- View, clear, and reset high scores
- Responsive layout for mobile and desktop

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage

---

## How to Use
1. Open the application in your browser.
2. Click the **Start** button to begin the quiz.
3. Select an answer for each question.
4. Wrong answers subtract time from the timer.
5. When the quiz ends, enter your initials to save your score.
6. View high scores or restart the quiz.
